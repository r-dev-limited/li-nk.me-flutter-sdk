## Installation

```bash
flutter pub add flutter_linkme_sdk
```

Or declare it manually:

```yaml
dependencies:
  flutter_linkme_sdk: ^0.1.0
```

The plugin links against:
- LinkMe Android SDK `0.1.2` from JitPack (`com.github.r-dev-limited:li-nk.me-android-sdk`)
- LinkMeKit `0.1.2` via Swift Package Manager
